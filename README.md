## Usage

To run this app, follow these steps:

1. Clone the repository:

   ```bash
   git clone https://github.com/Navedms/AskYourNationApp.git
   cd AskYourNationApp
   ```

2. Install dependencies:

   ```bash
   npm install
   ```

3. Start the development server:
   ```bash
   npm start
   ```
